# 🗺️ 地图大比拼 · 中国省份拼图

**中国省份拼图游戏，边玩边认地图。**

一款为中国小学生设计的省份地理拼图小游戏：34 个省级行政区按真实省界拼合，配合三档难度的知识问答，让孩子在动手拼图的过程中轻松记住每个省的位置、简称、省会和特色。

「地图大比拼」是一款面向小学生的中国地理启蒙小游戏。核心玩法是将 34 个省级行政区按真实地理轮廓拖拽拼合，还原完整中国地图。游戏提供多档拼图难度适应不同年龄，并内置三档知识问答——从最基础的"省的省会在哪"，到混入简称、别称的进阶题，再到结合景点、美食、文化、历史典故的趣味题，每局随机生成 20 道混合题并给出百分制成绩。另支持分区练习、画面缩放与平移、计时与最佳成绩记录等功能。整体以轻量、无广告、可离线运行为设计原则，适合家庭亲子共玩或孩子自主练习，把抽象的行政区划知识变成可触摸、可互动的游戏体验。

---

## ✨ 功能特色

- 🧩 **真实省界拼图**：基于公开地理数据绘制的 34 个省级行政区真实轮廓
- 🎚️ **三档拼图难度**
  - 🐵 **猴子方块**：彩色方块认位置，适合低龄入门
  - 👧 **小孩真实标字**：真实形状 + 省名标注，建立形状与名称对应
  - 🧔 **爸爸真实不标字**：真实形状无标注，自我检验用
- 📝 **知识问答三难度**：小学生 / 幼儿园 / 托儿所，题型从省会归属到省情特色层层递进
- 🗺️ **分区练习**：可按华北、东北、华东等七大地理分区单独练习
- ⏱️ **计时 + 最佳成绩**：每局记录用时，激励反复挑战
- 🔍 **缩放与平移**：手机双指、PC 滚轮缩放，单指/左键拖拽平移
- 📊 **数据表**：内置省级行政区数据页，含省名别称、省会、省会别称、面积、特色

## 🎮 怎么玩

**方式一：本地直接打开**
双击 `map_battle.html` 即可在浏览器中游玩，无需联网、无需安装。

**方式二：部署到网页（推荐，方便分享）**
将本仓库部署到 GitHub Pages 后，会得到一个公开网址，发给家人朋友即可直接打开。步骤见下方「部署说明」。

---

## 🚀 部署到 GitHub Pages（简单流程）

1. 在 GitHub 新建仓库（如 `map-battle`）
2. 把本文件夹里的 `map_battle.html` 上传到仓库
3. 进入仓库 **Settings → Pages**
4. Source 选 **Deploy from a branch**，Branch 选 **main**，文件夹选 **/ (root)**，保存
5. 一两分钟后访问 `https://你的用户名.github.io/map-battle/map_battle.html` 即可

> 之后每次更新 `map_battle.html`，网页内容会自动同步。

---

## 📁 文件说明

| 文件 | 说明 |
|------|------|
| `map_battle.html` | 游戏本体（单文件，可直接打开/部署） |
| `gen_map.py` | 源码生成器（用 Python 重新生成 HTML，方便二次修改数据） |

如需修改省份数据（特色、别称等），编辑 `gen_map.py` 中的 `META` / `CAP_FEAT` 等字典后运行 `python3 gen_map.py` 即可重新生成 `map_battle.html`。
